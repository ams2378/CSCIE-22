Problem Set 0

Note

This assignment is due as soon as possible.

  • Task 1: send us an e-mail
  • Task 2: review course policies
  • Task 3: install the necessary software
  • Task 4: practice submitting a file
  • Task 5: check your account settings on Canvas

Task 1: send us an e-mail

Send an e-mail to the course account (cscie22@fas.harvard.edu) from whatever e-mail account you read most frequently.

Please use the subject line “Problem Set 0”

Please include the following information:

  1. your full name
  2. the nickname or other name that you would like us to use for you
  3. your credit status (undergraduate, graduate or non-credit) and whether you are taking the course via the distance-ed option
  4. your hometown
  5. your current educational or work status
  6. a brief description of your background, including your prior programming experience
  7. a brief description of why you’re taking this course

Important: Although we do eventually read through these emails, it can take us awhile to do so. Therefore, if you have a question or other matter that needs immediate attention, please send a separate e-mail to the course account.

Task 2: review course policies

As your next task, please review the policies of the course, which are outlined in the syllabus. In particular, please review carefully the policies on academic conduct. After you have read over these policies, feel free to let us know if you have any questions.

Task 3: install the necessary software

In your work for this course, you will need to have access to the following programming tools:

  • the Java development kit (JDK). Although most computers already have the necessary software to run Java programs, they don’t all have the necessary software to develop new Java programs, which is why you may need to install the JDK.
  • an IDE such as DrJava, which you will use to develop your programs. (Note: You’re welcome to use a different IDE – or none at all – when developing your programs. However, we recommend DrJava for this course, and it is the only IDE for which we will be able to offer support.)

Task 4: practice submitting a file

We will be using Canvas for homework submissions. To avoid any last-minute problems when submitting the first homework, you may want to practice submitting a file as follows:

  • Use a text editor to create a plain-text file, naming the file ps0_test.txt. The actual contents of the file don’t matter, as long as it contains some text.
  • Go to the page for submitting assignments (logging in as needed using your HarvardKey).
  • Click the link for ps0.
  • Click on the Submit Assignment button near the upper-right corner of the screen.
  • Use the Choose File button to select your ps0_test.txt file. (If the assignment had multiple files, you would continuing selecting the remaining files. Note that you must submit all of the files for a given assignment at the same time. If you need to resubmit a file for some reason, you should also resubmit all of the other files from that assignment.)
  • Click on the Submit Assignment button.

After submitting the file, you should check your submission carefully. In particular, you should:

  • Check to make sure that you have a green checkmark symbol labeled Turned In! in the upper-right corner of the submission page (where the Submit Assignment button used to be), along with the names of all of the files that you are submitting.
  • Click on the link for the file to download it, and view the downloaded file so that you can ensure that you submitted the correct file.

We will not accept any files after the fact, so please check your homework submissions carefully!

Task 5: check your account settings on Canvas

We will also be using Canvas for announcements. To ensure that you are receive all announcements as soon as they are sent, you should do the following:

  • Go to Canvas (logging in as needed using your HarvardKey).
  • Click the link labeled Account in the left-hand navigation bar, and then click Settings.
  • Check that your Default Email is one that you check on a regular basis. If necessary, find the section of the page labeled Ways to Contact and add an email address. If you have more than one email address listed, make sure that the star appears next to one that you check regularly.
  • Next, click the link labeled Account and then click Notifications.
  • In the Notification Preferences table, find the row labeled Announcement, and click the checkmark symbol in the column from that row that corresponds to the email address that you check regularly. This will ensure that you are notified right away of all new announcements.
